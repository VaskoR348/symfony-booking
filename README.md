# Symfony Booking Application

A booking management system built with Symfony 4.4 framework.

## Features

- User authentication and registration
- Business owner management
- Listing and booking system
- Review and rating functionality
- Blog with comments
- Wishlist support

## Installation

```bash
composer install
php bin/console doctrine:migrations:migrate
```

## Requirements

- PHP >= 7.1.3
- MySQL/MariaDB
- Symfony 4.4
